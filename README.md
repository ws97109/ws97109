## 精選專案

### 高速公路智慧交通震波預警 AI 決策系統

[![GitHub](https://img.shields.io/badge/專案連結-181717?style=for-the-badge&logo=github)](https://github.com/ws97109/Highway_traffic)
![Status](https://img.shields.io/badge/Status-獲獎專案-gold?style=for-the-badge)

> 首創將地震學震波理論應用於交通流分析，結合 MT-STNet 深度學習與傳統交通理論，實現毫秒級即時震波檢測與智慧預警

#### 創新突破

**理論創新**
- 首創將地震學震波傳播理論應用於交通分析
- 基於 Indiana 州研究，後向震波速度 4.2 mph
- 87% 檢測準確率，三級警報系統

**技術架構**
- **前端**：Next.js 15 + TypeScript 響應式應用
- **後端**：FastAPI 高效能 API（響應時間 < 200ms）
- **AI 核心**：TensorFlow + MT-STNet 深度學習引擎

#### 核心技術

**MT-STNet 多任務神經網路**
- 同步預測：交通流量、速度、密度
- 支援 12 個歷史時步預測未來 12 個時步
- 整合 17 種基準模型：AGCRN、DCRNN、Graph-WaveNet 等

**震波檢測演算法**
- 輕微警報：速度下降 6-18 km/h
- 中等警報：速度下降 18-30 km/h  
- 嚴重警報：速度下降 30+ km/h

**RAG 智能助手**
- 學習道路幾何資料與即時數值
- 提供情境化建議與決策支援

#### 系統功能

**駕駛者服務**
- 即時震波預警與到達時間預測
- 智慧路線規劃與替代方案
- 出發時間優化建議
- 油耗與時間成本估算

**管理者功能**
- 全路網即時監控
- AI 驅動決策支援系統
- 預防性交通管制策略
- 大屏數據中心

#### 性能指標

| 指標 | 數值 |
|------|------|
| 震波檢測延遲 | < 5 秒 |
| API 響應時間 | < 200ms |
| 交通流量預測 MAE | 12.3 veh/5min |
| 車速預測 MAE | 3.2 km/h |
| 密度預測 MAE | 8.9 veh/km |

**競賽成果**：高公局國道智能競賽亞軍

**技術標籤**：時空預測 • 圖神經網路 • 震波檢測 • 智慧交通 • 深度學習

### 強化學習德州撲克 AI 系統

[![GitHub](https://img.shields.io/badge/專案連結-181717?style=for-the-badge&logo=github)](https://github.com/ws97109/poker_ai)

#### 核心特色

**多人對戰系統**
- 支援 2-7 個玩家的多人德州撲克對戰
- 靈活的遊戲配置系統
- 完整的遊戲邏輯實現

**智慧探索策略**
- ε-greedy 探索策略
- 探索率從 0.9 自適應衰減至 0.1
- 平衡探索與利用的動態調整

**完整遊戲邏輯實現**
- 牌型評分系統與識別
- 下注輪次管理（翻牌前、翻牌、轉牌、河牌）
- 分池機制與位置管理
- 全押與邊池處理

**豐富的訓練視覺化**
- 勝率追蹤與趨勢分析
- 獎勵變化曲線
- 策略演化記錄
- 探索率衰減監控

**網頁遊戲介面**
- 提供即時對戰體驗
- 視覺化遊戲狀態展示
- AI 決策過程透明化

**使用技術**：Python • TensorFlow • 深度強化學習 • DQN • HTML/JavaScript • 博弈理論

### 網路行為研究

[![GitHub](https://img.shields.io/badge/專案連結-181717?style=for-the-badge&logo=github)](https://github.com/ws97109/Internet-behavior)

深度探討不同群體網路使用模式與霸凌現象關聯的學術研究專案，運用多變量統計分析技術建立行為預測模型。

#### 核心特色

**基於台灣傳播調查資料庫的實證分析**
- 樣本規模：672 位參與者
- 資料來源：Taiwan Communication Survey Database
- 研究對象：不同年齡層與背景的網路使用者

**運用廣義關聯圖（GAP）技術**
- 建立網路使用者行為分類模型
- 識別不同使用模式
- 群體特徵分析

**採用主成分分析（PCA）和因素分析**
- 探討行為特徵與模式
- 降維與特徵提取
- 潛在因子識別

**透過典型相關分析（CCA）**
- 建構網路行為與霸凌傾向的因果關係模型
- 多變量關聯性分析
- 預測模型建立

**提供實證基礎**
- 網路素養教育政策制定
- 反霸凌策略數據支持
- 教育機構政策參考
- 社會問題研究工具

**使用技術**：Python • 隨機森林 • 統計分析 • PCA • 因素分析 • 典型相關分析 • 數據視覺化

- 圖像尺寸調整（多種預設比例）
- 生成步數設定（平衡速度與品質）
- 引導係數調整

**支援批量生成**
- 一次產出多張高品質圖像
- 自動編號與儲存
- 效率提升

**基於 Gradio 的友善 Web 介面**
